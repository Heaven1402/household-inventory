# Household Appliance Inventory System

This project is a web-based application developed using Next.js and MySQL.

## Features

- Add new appliances
- Search appliances by serial number
- Update appliance details
- Delete appliances
- Relational database (Users & Appliances)

## Technologies Used

- Next.js
- MySQL
- Node.js

## How to Run

1. Install dependencies:
   npm install

2. Run the project:
   npm run dev

3. Open:
   http://localhost:3000
